# 📚 BookVerse

**Live Demo:** https://book-verse-weld.vercel.app/  
**GitHub Repository:** https://github.com/kumarsunny-nitian/BookVerse  

BookVerse is a full-stack online bookstore application that allows users to browse, manage, and purchase books with secure authentication and a smooth user experience.

---

## 🚀 Features

- 🔐 JWT-based Authentication (Login / Register)
- 📖 Browse and search books
- 🛒 Add to cart & purchase functionality
- 📦 Order management system
- 🧑 User profile management
- 🌐 Responsive user interface

---

## 🛠️ Tech Stack

### 💻 Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### 🔙 Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Token)
- bcrypt.js

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/kumarsunny-nitian/BookVerse.git
cd BookVerse
```

### 2️⃣ Install dependencies

Frontend:
```bash
cd Frontend
npm install
npm run dev
```

Backend:
```bash
cd Backend
npm install
npm start
```
